# IWebUser

The interface is both implemented by `WebUser` and `TokenWebUser`.
It is used to set the `webuser` property in `Controller`.


## Identity

```csharp
/// <summary>
/// true if user connected
/// </summary>
bool Identity { get; }
```


## Id

```csharp
        /// <summary>
        /// The user id key
        /// </summary>
        Guid Id { get; }
```


## Login

```csharp
        /// <summary>
        /// The user login
        /// </summary>
        string Login { get; }
```


## Mail

```csharp
/// <summary>
```

## FullName

```csharp
/// <summary>
```
